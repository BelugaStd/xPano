Design system direction: Apple-inspired commercial workbench with layered rounded surfaces, restrained dark-neutral palette, clear hierarchy, subtle motion-ready components, and precise SVG iconography.
Design DNA blend: Mobile-First App with supporting cues from 3D / Interactive Site and Operational Dashboard.

Visual rules:
- Original design, not a copy of any existing brand or website.
- Realistic app interface that can be implemented in Qt Quick or React/CSS.
- Clear information hierarchy, readable UI labels, coherent component system.
- Use consistent typography, spacing, border radius, icon style, and color tokens.

Layout requirements:
- Use a native-feeling portrait mobile layout optimized for monitoring rather than full editing.
- Top area shows current project and global status.
- Primary content shows latest left/right fisheye preview stacked vertically, extraction progress, and current track name.
- Secondary content is collapsed into tabs: Tracks, Log, Output.
- Include a sticky bottom action bar with pause, stop, and open output controls.
- Use large touch targets, readable text, and strong vertical rhythm.

Output: one single mobile portrait UI screen only. Do not include desktop views, annotations, browser chrome, or multiple viewport mockups.
